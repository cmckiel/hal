# Pinout

The HAL currently caters to the stm32f446re specific dev board. This will be fixed in a later release.

| Pin  | Function |
| ---- | -------- |
| PB6  | UART1 TX |
| PB7  | UART1 RX |
| PA2  | UART2 TX |
| PA3  | UART2 RX |
| PB8  | I2C1 SCL |
| PB9  | I2C1 SDA |
| PA8  | PWM Channel 1 |
| PA9  | PWM Channel 2 |
| PA10 | PWM Channel 3 |
| PA11 | PWM Channel 4 |
| PA5  | LED      |
