void hal_pwm_set_frequency ( uint32_t  pwm_frequency_hz)

Changes the base PWM frequency shared by all channels. The duty-cycle ratio of every enabled channel is preserved by rescaling its compare register.

Parameters

Initializes the hardware timer (TIM1) for PWM output at the given base frequency. Must be called once before initializing any channel. All channels share this frequency.

Parameters
