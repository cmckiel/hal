# Pinout

The HAL currently caters to the stm32f446re specific dev board. This will be fixed in a later release.

| Pin  | Function |
| ---- | -------- |
| PA9  | UART1 TX |
| PA10 | UART1 RX |
| PA2  | UART2 TX |
| PA3  | UART2 RX |
| PB8  | I2C1 SCL |
| PB9  | I2C1 SDA |
| PA8  | PWM1     |
| PA5  | LED      |
